TradeApp

A simulated trading dashboard powered by real-time sentiment signals and portfolio tracking.

Overview

TradeApp simulates market behavior by converting scraped financial tweets into structured trading signals. It allows users to execute trades, monitor positions, and track performance in a dynamic environment.

Database Architecture

Built using PostgreSQL with SQLAlchemy ORM and Alembic migrations.

tweets — Stores scraped Twitter/X data (signal source)

tweet_sentiments — AI sentiment + stock extraction

user_settings — User configs & trading preferences

Run Locally

cd server
uv sync
uv run uvicorn main:app

Database Implementation Plan

Supabase-backed PostgreSQL system designed to persist scraped tweets, AI-generated signals, and user configurations with duplicate prevention and migration support.

Key components: migrations (Alembic), local Docker Postgres, and upsert-based ingestion pipeline.
